## �� 平台概述

**IIPlatform** 是一套專為工業物聯網設計的綜合性管理平台，整合了設備監控、數據分析、AI 智能、自動化控制等全方位功能。平台採用現代化的微服務架構，支援多種通訊協定，為製造業提供完整的數位化轉型解決方案。

### �� 平台定位
- **工業 4.0 核心平台**：支援智慧製造與數位化轉型
- **多協定整合**：統一管理不同通訊協定的設備
- **AI 驅動分析**：智能預測與決策支援
- **即時監控**：24/7 設備狀態與效能監控

## 🚀 快速開始

### 環境需求
- Python 3.9+
- Node.js 16+
- PostgreSQL 12+
- Redis 6+

### 快速安裝
```bash
# 克隆專案
git clone <repository-url>
cd IIPlatform

# 後端設定
cd backend/app
pip install -r requirements.txt
uvicorn main:app --reload

# 前端設定
cd frontend
npm install
npm start
```
